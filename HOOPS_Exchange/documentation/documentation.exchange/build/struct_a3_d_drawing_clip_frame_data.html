Field Documentation

A3DBool A3DDrawingClipFrameData::m_bClipBound

Clip view geometry with the boundary.

A3DBool A3DDrawingClipFrameData::m_bDrawBound

Draw the boundary.

A3DEDrawingClipFrameType A3DDrawingClipFrameData::m_eType

Type of current clipping frame (see A3DEDrawingClipFrameType).

A3DCrvBase* A3DDrawingClipFrameData::m_pCurve

Closed curve of clipping if m_eType is kA3DDrawingClipFrameTypeUserDefine.

A3DDomainData A3DDrawingClipFrameData::m_sRectangularBox

BoundingBox of clipping if m_eType is kA3DDrawingClipFrameTypeRectangular.
Coordinate are in local coordinate space.

A3DUns16 A3DDrawingClipFrameData::m_uiStyle

Style of the boundary if m_bDrawBound is set to true.

A3DUns16 A3DDrawingClipFrameData::m_usStructSize

Reserved; must be initialized with A3D_INITIALIZE_DATA.
