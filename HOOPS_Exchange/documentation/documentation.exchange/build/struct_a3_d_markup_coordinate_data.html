Detailed Description

Markup text with additional data. Use A3DMarkupTextGet to access to markup text data.

Field Documentation

A3DBool A3DMarkupCoordinateData::m_bType3D

Specifies if it's a 2D or a 3D coordinate; if it's the latter, m_dLitDimensionedZis is initialized.

A3DDouble A3DMarkupCoordinateData::m_dLitDimensionedX

Coordinate x.

A3DDouble A3DMarkupCoordinateData::m_dLitDimensionedY

Coordinate y.

A3DDouble A3DMarkupCoordinateData::m_dLitDimensionedZ

Coordinate z.

A3DMDTextPosition* A3DMarkupCoordinateData::m_psTextPosition

The relative position of the text to the attach of the markup.

See also
A3DMDTextPositionGetData
A3DUns16 A3DMarkupCoordinateData::m_usStructSize

Reserved; must be initialized with A3D_INITIALIZE_DATA.
