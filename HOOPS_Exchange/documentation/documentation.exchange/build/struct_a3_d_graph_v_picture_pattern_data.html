Detailed Description

VPicture Pattern Structure.

Version
2.0
Warning
m_pMarkupTess is a restricted version of kA3DTypeTessMarkup. The authorized entities are as follows:
  • Polyline
  • Triangles
  • Color
  • Line stipple
  • Points
  • Polygon
  • Line width
