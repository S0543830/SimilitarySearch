Detailed Description

Markup roughness.

defined in file A3DSDKMarkupSymbol.h
Values of m_eRoughType, m_eRoughMode, and m_eRoughContactArea define the Roughness symbol form.
The differents fields positions are placed all around the roughness symbol as we can see below.
