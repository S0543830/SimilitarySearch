CATIA V4 Reader

The CATIA V4 Reader provides full access to the Assembly, B-rep and PMI data stored within a CATIA V4 file. A file does not need to have a particular extension to be read correctly by HOOPS Exchange however the file extensions of CATIA V4 files that you can expect to be loaded by HOOPS Exchange are:

  • .DLV
  • .EXP
  • .MODEL
  • .SESSION

Our readers are updated to the latest released version within 90 days of the release of the authoring software. You can see the exact version supported in the current release here.

Assembly

Incremental load is supported by our CATIA V4 reader. Datum planes and construction geometry are also supported and accessed through the assembly tree.

B-rep

B-rep support within CATIA V4 is pretty basic and so the list of different types of geometries is small. HOOPS Exchange provides evaluators which allow you to direct query all the geometry types available in CGM. You can additionally ask HOOPS Exchange to approximate any specific geometry and provide it in a standard NURBS form.

The list of surface types available in CATIA V4 is:

  • B-spline
  • Cylinder
  • Extrusion
  • Plane
  • RuledSurface
  • SurfOfRevolution

The list of curve types available in CATIA V4 is:

  • B-spline
  • Circle
  • Conic (ellipse)
  • Conic (hyperbola)
  • Conic (parabola)
  • Line

Visualization

The visualization data for this format is generated by the HOOPS Exchange tessellation engine.

Views

This concept is not supported by CATIA V4.

PMI

CATIA V4 has very basic PMI support which consists solely of support for a 3D Note. This is supported by HOOPS Exchange.
